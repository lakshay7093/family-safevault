# family-safevault
Family SafeVault is a secure digital vault app for families to store and manage important documents in one safe place. It provides encrypted storage, role-based access, emergency access, and real-time live location tracking. The platform protects privacy, enables secure sharing and improves family safety through controlled access and GPS monitoring
